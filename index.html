❀ HTML Tags Cheat Sheet ❀

Here's a list of commonly-used HTML tags!

HTML Tag Name Description
<div> Div Section The <div> HTML element is the generic container for flow content.
<h1> to <h6> Headings The <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.
<p> Paragraph The <p> tag defines a paragraph.
<img> Image The <img> tag is used to embed an image in an HTML page.
<a> Anchor The <a> tag defines a hyperlink, which is used to link from one page to another.
<strong> Bold Text The <strong> tag is used to define text with strong importance. The content inside is typically displayed in bold.
<em> Italic Text The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.
<span> Span Container The <span> tag is an inline container used to mark up a part of a text, or a part of a document.
<br> Line Break The <br> tag inserts a single line break. The <br> tag is useful for writing addresses or poems.
<q> Quotation The <q> tag defines a short quotation. Browsers normally insert quotation marks around the quotation.
<video> Video The <video> HTML element embeds a media player which supports video playback into the document.
<audio> Audio The <audio> tag is used to embed sound content in a document, such as music or other audio streams.
<ul> and <ol> Lists HTML lists start with <ul> for unordered lists, and <ol> for ordered lists. List items start with <li>.
<table> Table An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.
